# Project Overview
This project is an interactive dashboard developed using Microsoft Power BI to transform raw business data into meaningful insights. The dashboard helps analyze performance trends, track key metrics, and support decision-making through interactive visualizations. It focuses on presenting data in a simple and user-friendly format for better business understanding.


# Project Highlight
Built an end-to-end interactive dashboard for business data analysis
Used Power Query for data cleaning and preprocessing
Created custom DAX measures for KPI calculations
Designed dynamic visuals to compare performance across different categories
Added slicers and filters for interactive user exploration
Provided actionable insights from raw data through storytelling visuals


# Key Features
KPI cards for important metrics (sales, profit, orders, etc.)
Interactive slicers for filtering data by category, region, and time
Trend analysis with charts and graphs
Category-wise and region-wise performance comparison
Clean dashboard layout for easy interpretation
Drill-down capability for detailed analysis
